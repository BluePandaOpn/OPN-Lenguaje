# Inicio Rápido  OPN en 5 Minutos

 [Volver al README](../README.md)

Bienvenido a OPN. Aprende lo básico en 5 minutos.

---

##  Paso 1: Tu Primer Programa

Crea **hola.opn**:
````opn
print("¡Hola, OPN!");
````

Ejecuta:
````ash
python opn2.py hola.opn
````

 **¡Felicidades!** Has ejecutado tu primer programa OPN.

---

##  Paso 2: Variables

Crea **variables.opn**:
````opn
var edad = 25;
var nombre = "Carlos";
var altura = 1.75;

print("Nombre: " + nombre);
print("Edad: " + edad);
print("Altura: " + altura + " metros");
````

Ejecuta: python opn2.py variables.opn

---

##  Paso 3: Funciones

````opn
function suma(a, b) {
    return a + b;
}

print(suma(10, 20));
````
